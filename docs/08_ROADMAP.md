# Roadmap

この文書は、BARBER HUBの今後の方向性を「近い予定」「次の段階」「将来構想」「保留」に分けて整理します。固定日付の約束ではありません。

## 近い予定

- 既存トップ画面の安定化。
- 店舗ディレクトリの本番接続と検索体験の安定化。
- Supabase migration承認フローの運用確認。
- 公開前セキュリティ確認。
- Back Room、Snap、求人、Q&A、プロフィールなど既存機能の破壊防止。
- UI文言の整理と説明量の削減。
- 3MIN NEWSを、仕事・提案・会話に使えるWORK / STYLE / TALK相当の構成で運用し、若い男性理容師が毎日確認しやすい内容へ改善する。

## 次の段階

- 店舗CSV取込管理画面。
- 福岡・九州の店舗データ追加。
- 店舗データの追加・修正・閉店・重複対応の運用設計。
- 店舗データCSV取込の手動workflowまたは管理画面の検討。
- 店舗認証申請の審査フロー設計。
- 管理者ロール、管理画面、操作ログの設計。
- 問い合わせ、広告、協賛の運用整理。
- Back Roomのモデレーション導線の検討。

## 将来構想

- 業界ディレクトリ拡張。
- メーカー、ディーラー、学校、組合などの掲載。
- 公式認証や協賛商品の設計。
- 広告商品の拡張。
- 業界ニュースやイベント情報の運用効率化。
- メンズファッション、メンズヘア・ヘアケア、メンズスキンケア、音楽・BGM、エンタメ、主要スポーツを、著作権・承認フロー・AI費用を守りながら3MIN NEWSで継続改善する。
- 地域別・カテゴリ別の検索体験改善。
- パートナー向け掲載管理。
- 各既存機能の説明文削減・UI整理。
- 公開前セキュリティ・法務・運営確認。

## 保留・追加しない方針

- 店舗口コミ。
- 店舗星評価。
- 店舗ランキング。
- 本番seedの自動投入。
- mainマージ直後の本番DB完全自動変更。
- 承認なしの危険migration適用。
- 実在店舗や企業情報の無断登録。
- 店舗PRにメーカー、ディーラー、学校、組合の実装を混ぜること。
- 3MIN NEWSを芸能ゴシップサイトやスポーツ速報サイトにすること。
- 3MIN NEWSでAI下書きを自動公開すること。

## 要確認

- FREEプランでのバックアップ運用。
- 管理画面の権限モデル。
- 審査担当者と開発者の権限分離。
- 広告・協賛の契約、素材、停止ルール。
- 店舗データの情報源と利用許諾。
