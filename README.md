# BARBER HUB v0.1

理容師が毎朝スマホで開きたくなる、BARBER HUBのMVPトップページです。

## 作ったもの

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- スマホファーストのトップページ1枚
- AI編集部、メイン記事ローテーション、カテゴリー、Feed、THANKS、Q&A、講習会アーカイブ、学生・求人、投稿メニュー

## 起動手順

最初の1回だけ、パッケージを入れます。

```bash
pnpm install
```

そのあと、開発サーバーを起動します。

```bash
pnpm dev
```

ブラウザで表示されるURLを開くと確認できます。通常は以下です。

```text
http://localhost:3000
```
